# examples-stm32

This repository contains source code of Embedded Control Programs (as Examples) for MCU STM32.

**STM32F103**

(ARM Cortex-M3 32-bit RISC)

- Hardware Platform
  - BluePill
- Software Platform
  - IDE
    - Eclipse
  - Compiler
    - ARM GNU GCC
  - Language
    - C
  - Frameworks
    - STM32-SPL
  - Operation System
    - BareMetal (no OS)
    - FreeRTOS

**STM32F107**

(ARM Cortex-M3 32-bit RISC)

- Hardware Platform
  - WaveShare Core107V
- Software Platform
  - IDE
    - Eclipse
  - Compiler
    - ARM GNU GCC
  - Language
    - C
  - Frameworks
    - STM32-SPL
  - Operation System
    - BareMetal (no OS)
    - FreeRTOS
